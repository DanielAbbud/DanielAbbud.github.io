Poods Store — v6 (sabores opcionais)
====================================

- Visual moderno + WhatsApp direto.
- Produtos podem ter sabores (se `flavors` existir, aparece um seletor no card).
- O sabor escolhido é incluído na mensagem do WhatsApp.

Como configurar:
1) Troque o número: script.js -> const WHATSAPP_NUMBER = "55DDDNUMERO"
2) Adicione/edite produtos no array PRODUCTS
3) Para sabores, inclua: flavors: ["Morango","Chocolate"]
4) Imagens em: assets/products/
